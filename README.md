# ApplyForge - ATS-Friendly Resume Builder

ApplyForge is a Streamlit-based resume builder that helps generate ATS-friendly resume drafts tailored to specific job descriptions.

The project analyzes job postings, extracts relevant skills, matches them against an evidence-based candidate profile, and generates resume bullet recommendations without unsupported claims.

## Features

- Job description input
- Skill extraction
- Candidate profile matching
- Evidence-based resume bullet selection
- Honesty guard for unsupported claims
- ATS text preview
- Markdown resume draft generation

## Tech Stack

- Python
- Streamlit
- Pydantic
- Jinja2
- python-docx
